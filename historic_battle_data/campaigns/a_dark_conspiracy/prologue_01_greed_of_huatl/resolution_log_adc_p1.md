## TURN 1: The Trap is Closing
**Under the Cover of Darkness**: Rolled 2. Visible distance is 2 * 3 = 6" (15cm).

### Bretonnians
* *The Peasants* **March** at 20 cm/turn. Being 60 cm from the middle of the temple to the escape zone, they need three full turns of marching, so they spend the first two looting. 
* *The Knights* face south, ready to run or charge, but remain looting.
* **Securing the Loot:** 4 markers generate 4x25 VPs.

### Lizardmen 
* *Northern Jaw* moves to the western front, **Marching** through the dense vegetation (**Soft Cover**). No visibility to shoot.
* *Western Jaw Skinks* **March** to get closer to the *Looters of the Duc*. Three are closer than 6", the rest close enough to shoot.
* *Eastern Jaw* moves through the jungle to shoot the *Vanguard of the Duc*.
    * **To Hit:** Base BS3 (4+) + **Long Range** (+1) + **Move and Shoot** (+1) + **Multiple Shots** (+1) = Needing 7+ -> Roll 16 darts: 5 chances -> 4 hits.
    * **To Wound:** S3 vs T3 = Needing 4+ -> Rolls: 5, 3, 3, 3 -> 3 wounds.
    * **Armour Save:** Base 2+ (**Heavy Armour**, **Shield**, **Mounted**, **Barding**) -> Roll: 3 -> Saved.
    * **Poison Rule:** 4 darts needing 6+ to hit -> Rolls: 5, 6, 3, 5 -> 1 auto-wound.
    * **Armour Save:** Base 2+ -> Roll: 2 -> Saved.
* *Western Jaw* shoots poisoned darts over the *Looters of the Duc*:
    * **To Hit:** Base BS3 (4+) + **Long Range** (+1) + **Move and Shoot** (+1) + **Multiple Shots** (+1) = Needing 7+ -> Roll 14 darts: 4 chances -> 1 hit.
    * **To Wound:** S3 vs T3 = Needing 4+ -> Roll: 4 -> 1 wound.
    * **Armour Save:** Base 2+ -> Roll: 6 -> Saved.
    * **Poison Rule:** 6 darts needing 6+ to hit -> Rolls: 2, 5, 6, 3, 4, 1 -> 1 auto-wound.
    * **Armour Save:** Base 2+ -> Roll: 3 -> Saved.

## TURN 2: Rainforest of Arrows

**Under the Cover of Darkness**: Rolled 6. Visible distance is 6 * 3 = 18" (45cm).

### Bretonnians
* *The Duc* remains looting. *The Hunters* declare a **Charge** to the skinks of the south-east, who react by **Fleeing**. **Failed Charge** forces normal movement.
* *The Duc's Vanguard* **Charge** the *Bronze Shields*. 5 knights and 5 warhorses strike. The *Gallant* issues a **Challenge** to the Scar-Veteran. Combat is resolved in **Initiative** order:

1. **The Vanguard Gallant attacks Scar-Veteran Quntael (Challenge):**
    * **To Hit:** WS4 vs WS5 = Needing 4+ -> 2 attacks: 3, 4 -> 1 hit. 
    * **To Wound:** S5 (S3 + **Lance**) vs T4 = Needing 4+ -> Roll: 4 -> 1 wound. 
    * **Armour Save:** Quntael Base 4+ (Scaly skin, Light armour) modified by -2 (S5) = 6+ Save -> Roll: Fail -> 1 wound suffered.
2. **Scar-Veteran Quntael strikes back at the Gallant:**
    * **To Hit:** WS5 vs WS4 = Needing 3+ -> 4 attacks: 5, 4, 2, 3 -> 3 hits. 
    * **To Wound:** S5 vs T3 = Needing 2+ -> Rolls: 6, 4, 4 -> 3 wounds. 
    * **Armour Save:** Gallant Base 2+ modified by -2 (S5) and -1 (*Biting Blade*) = 5+ Save -> Fails 2 -> The Gallant dies. Due to **Overkill**, the additional wound counts toward **Combat Resolution**.
3. **The Vanguard Knights attack the Saurus Warriors:**
    * **To Hit:** WS4 vs WS3 = Needing 3+ -> 4 attacks: 2, 4, 3, 6 -> 2 hits. 
    * **To Wound:** S5 vs T4 = Needing 3+ -> Rolls: 5, 5 -> 2 wounds. 
    * **Armour Save:** Saurus Base 4+ (Scaly skin, Shield, Hand Weapon) modified by -2 (S5) = 6+ Save -> Rolls: Fail -> Two Saurus die.
4. **The Vanguard Warhorses attack the Saurus Warriors:**
    * **To Hit:** WS3 vs WS3 = Needing 4+ -> 5 attacks: 5, 4, 6, 3, 3 -> 3 hits. 
    * **To Wound:** S3 vs T4 = Needing 5+ -> Rolls: 6, 2, 1 -> 1 wound. 
    * **Armour Save:** Saurus Base 4+ (Unmodified by S3) -> Roll: Pass -> 0 wounds suffered.

### Lizardmen
* *Eastern Jaw* passes **Cold-Blooded Leadership** test -> 3D6: 1, 2, 6 (Result 3) < Ld 5 -> Rallies.
* Skink Priest Xiin casts *The Comet of Casandora* (11+) using 3 **Power Dice** -> Rolls: 3, 6, 3 (Result 12) -> Success. 
* Bretonnians attempt to **Dispel** using 2 dice -> Rolls: 4, 3 (Result 7) -> Fail. Target: center of the *Looters of the Duc*.

> **[Archivist's Note - Rule Citation]:** *The Comet of Casandora* can be cast upon any fixed point. Place a marker. Roll a D6 at the start of each player's turn. 1-3: Place another marker. 4-6: The spot is struck. All units within D6" multiplied by the number of markers take 2D6 Strength 4 hits.

* *Western Jaw* throws poisoned darts into the *Looters of the Duc*. 
    * **To Hit (Short Range):** Base BS3 (4+) + **Multiple Shots** (-1) = Needing 5+ -> 10 darts: 5, 1, 5, 3, 1, 2, 3, 3, 2, 5 -> 3 hits. 
    * **To Hit (Long Range):** Base BS3 (4+) + **Multiple Shots** (-1) + **Long Range** (-1) = Needing 6+ -> 10 darts: 5, 5, 1, 6, 5, 3, 5, 4, 5, 1 -> 1 hit (Auto-wound due to **Poison**).
    * **To Wound (Short Range):** 3 hits of S3 vs T3 = Needing 4+ -> Rolls: 3, 4, 5 -> 2 wounds. 
    * **Armour Save:** Knights base 2+ -> Rolls: 1, 4, 6 -> One fail -> One Knight dies.
* *Northern Jaw* shoots *Looters of the Duc*. 
    * **To Hit:** Base BS3 (4+) + **Move and Shoot** (-1) + **Long Range** (-1) + **Multiple Shots** (-1) = Needing 7+ -> Roll: 3 hits. 
    * **To Wound:** S3 vs T4 = Needing 4+ -> All miss.
* Combat continues between *Bronze Shields* and *The Duc's Vanguard*. Combat is resolved in **Initiative** order:

1. **Scar-Veteran Quntael attacks the Vanguard Knights:**
    * **To Hit:** WS5 vs WS3 = Needing 3+ -> Rolls: 2, 3, 6, 5 -> 2 hits. 
    * **To Wound:** S5 vs T3 = Needing 2+ -> 2 wounds. 
    * **Armour Save:** Knights Base 2+ modified by -2 (S5) and -1 (*Biting Blade*) = 5+ Save -> Fails 1 -> One Knight dies.
2. **The Vanguard Knights attack Scar-Veteran Quntael:**
    * **To Hit:** WS4 vs WS5 = Needing 4+ -> 1 attack: 5 -> 1 hit. 
    * **To Wound:** S3 vs T4 = Needing 6+ -> Roll 6 -> 1 wound. 
    * **Armour Save:** Quntael Base 4+ (Unmodified) -> Roll 6 -> 0 wounds suffered.
3. **The Vanguard Knights attack the Saurus Warriors:**
    * **To Hit:** WS4 vs WS3 = Needing 4+ -> 1 attack: 4 -> 1 hit. 
    * **To Wound:** S3 vs T4 = Needing 5+ -> Roll 5 -> 1 wound. 
    * **Armour Save:** Saurus Base 4+ (Unmodified) -> Roll: Pass -> 0 wounds suffered.
4. **The Vanguard Warhorses attack the Saurus Warriors:**
    * **To Hit:** WS3 vs WS3 = Needing 4+ -> Roll 1 -> Miss. 
5. **The Saurus Warriors strike at the Vanguard:**
    * **To Hit:** **Predatory Fury** (6 attacks). WS3 vs WS3 = Needing 4+ -> Rolls: 1, 3, 5, 3, 5, 1 -> 2 hits. 
    * **To Wound:** S4 vs T3 = Needing 3+ -> Rolls: 5, 4, 1, 4, 5, 4 -> 5 wounds. 
    * **Armour Save:** Knights Base 2+ modified by -1 (S4) = 3+ Save -> Pass all 5 saves.

## TURN 3: The Heavens Fall
**Under the Cover of Darkness**: Rolled 5. Visible distance is 15" (40cm). Rain begins.
* *The Comet of Casandora* triggers (Roll 5). 
    * **Impact:** 2D6 hits = 7 Strength 4 hits. 
    * **To Wound:** S4 vs T3 = Needing 3+ -> Rolls: 2, 6, 3, 1, 4, 3, 2 -> 4 wounds. 
    * **Armour Save:** Knight's 2+ modified to 3+ (S4) -> Two Knights fail and die. 
    * **Panic Test:** *Looters* test Ld -> Roll 11 (Fail) -> Re-roll via BSB: Roll 4 -> Success.

### Bretonnians
* *Looters of the Duc* **Charge** south into *Western Jaw*, destroying them in pursuit. 
* *Bowmen of Bordeleaux* shoot *Northern Jaw*. 
    * **To Hit:** Base BS3 (4+) + **Skirmishers** (-1) + **Soft Cover** (-1) = Needing 6+ -> Rolls: 5, 5, 4, 2, 4, 6 -> 1 hit. 
    * **To Wound:** S3 vs T2 = Needing 3+ -> Fails to wound.
* *Hunters* move into the forest to **March Block** the Saurus. 
* Bretonnians successfully **Dispel** the building Comet (Roll 11).
* **Securing the Loot:** 2 markers generate 2x25 VPs.

    * **To Wound:** S3 vs T3 = Needing 5+ -> Roll: Pass -> One Bowman dies.
